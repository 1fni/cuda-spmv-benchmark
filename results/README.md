# Benchmark Results

This directory contains benchmark results generated by `./scripts/run_all.sh`.

## Directory Structure

```
results/
├── raw/          # Raw benchmark outputs (TXT)
├── json/         # Structured results (JSON)
└── figures/      # Generated plots (PNG)
```

## Generating Results

```bash
# Quick verification (small matrix, ~2 min)
./scripts/run_all.sh --quick

# Full benchmarks (5000x5000 default)
./scripts/run_all.sh

# Custom matrix size
./scripts/run_all.sh --size=10000
```

## Generating Figures

```bash
pip install -r scripts/plotting/requirements.txt
python scripts/plotting/plot_results.py
```

## Note

Result files (*.txt, *.json, *.png) are gitignored to keep the repository clean.
Only the directory structure is tracked.
